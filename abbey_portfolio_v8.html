05

Hotspot, WiFi & Wired LAN — Three Deployment Modes

EduTrace runs entirely on the school's local network — no internet required at any point. The system supports three deployment configurations so schools can choose what fits their infrastructure. Teachers connect from any phone, tablet, or laptop browser on the same network.

  • Hotspot mode — admin laptop creates a WiFi hotspot; teachers connect their phones directly
  • WLAN mode — system runs on the school's existing WiFi router; any device on the network connects
  • Wired LAN mode — connected via Ethernet switch; highest stability for computer lab deployments
  • Auto-detects and displays the correct IP address for teachers to connect
  • QR code generated for instant teacher login — scan and go
  • No software installation needed on teacher devices — browser only
  • Works through NEPA blackouts — runs on laptop battery if needed

06

Automated Promotion, Condonation & Repeat

At the end of each academic session, EduTrace runs every student through the school's configured promotion rules — automatically deciding who is promoted, who is condoned, and who repeats. No manual calculations, no spreadsheet cross-checking. The admin reviews and confirms.

  • Configurable pass mark, minimum subjects passed, and subject failure limits
  • Per-class override rules — different criteria for JSS vs SSS classes
  • Condonation rule: promote borderline students who narrowly fail one subject
  • Bulk decision — entire class processed in one action
  • Individual override — admin can manually promote or hold back any student
  • All decisions logged in the audit trail with timestamps
  • Alumni records automatically created for graduating students

Technical Reference Notes
EduTrace — How It's Built Desktop application packaged as a Windows executable for easy school deployment. Runs a local web server accessible to all teachers on the same network via any browser. Supports three network configurations: laptop hotspot (no router needed), school WiFi, or wired LAN. All school data is stored locally — no cloud account or internet connection required at any point during normal operation.
Security & Data Protection Each installation is protected by a cryptographic licence key unique to that school. Staff accounts use industry-standard secure authentication with automatic lockout on repeated failed attempts. Sensitive credentials are encrypted at rest. All admin actions are logged with timestamps and are exportable as CSV. Automatic backups run on a configurable schedule — data is never lost.
This Portfolio Website Pure HTML/CSS/JS — zero runtime dependencies, deployable anywhere. All dynamic content (pricing, skills, testimonials) is driven by a single PORTFOLIO_DATA object — edit it to extend the site without touching HTML structure. Compatible with GitHub Pages, Netlify, Vercel, or any static file host.
